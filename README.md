# projectComPass
 Webprograming Team Project(team 4)

# 프로젝트 파일 구조

## css/
- `main.css`: 웹 페이지의 전반적인 스타일을 정의하는 CSS 파일입니다.

## js/
- `main.js`: 웹 페이지의 동적인 동작을 담당하는 JavaScript 파일입니다.

## images/
- `logo.png`: 웹 페이지에서 사용되는 이미지 파일입니다.

## pages/
- `index.html`: 웹 페이지의 메인 화면을 나타내는 HTML 파일입니다.
- `about.html`: 'About Us' 섹션과 관련된 HTML 파일입니다.
- `contact.html`: 'Contact Us' 섹션과 관련된 HTML 파일입니다.

## README.md
- 프로젝트에 대한 간략한 설명과 파일 구조를 정리한 문서입니다.
